# react-native-widget-stack

A recreation of Apple's iOS Widget Stack

## Installation

```sh
npm install react-native-widget-stack
```

## Usage


```js
import { multiply } from 'react-native-widget-stack';

// ...

const result = await multiply(3, 7);
```
